## GitHub Repository


This project implements a 3x3 board of randomly generated cards.
Each card contains:
- A point value (0–5)
- A color (red, green, blue)
- A cost in red, green, and blue tokens (0–4 each)
Clicking on a card replaces it with a new random card.
